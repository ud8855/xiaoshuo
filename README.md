# 小说写作技能库

为 Trae AI 系统打造的网络小说写作辅助技能集合。

## 技能列表

### 1. 男频小说写作助手 (male-novel-writer)
- **描述**: 专业的男频网络小说创作顾问，提供题材选择、角色设定、世界观搭建、剧情设计等专业指导
- **位置**: `.skills/male-novel-writer/skill.yml`
- **内容量**: 1777行
- **核心功能**:
  - 84+ 题材类型（都市流、玄幻流、穿越流、悬疑流、军事流、科幻流）
  - 28 种力量体系（修仙、异能、科技）
  - 48 种金手指设定
  - 48 种系统类型
  - 48 种世界观设定
  - 48 种人物设定
  - 48 种剧情脑洞
  - 36 种创新套路模板

### 2. 女频小说写作助手 (female-novel-writer)
- **描述**: 专业的女频网络小说创作顾问，提供题材选择、角色设定、世界观搭建、剧情设计等专业指导
- **位置**: `.skills/female-novel-writer/skill.yml`
- **内容量**: 1758行
- **核心功能**:
  - 96 种金手指设定
  - 96 种系统类型
  - 96 种世界观设定（含ABO等特色设定）
  - 96 种人物设定
  - 96 种剧情脑洞
  - 72 种创新套路模板
  - 48 个问答解决方案

## 使用说明

将 `.skills/` 目录下的技能文件夹放置到 Trae AI 系统的技能目录中即可使用。

## 技能特色

- **问答式交互**: 直接回答用户问题，不客套不废话
- **实战导向**: 提供具体可操作的写作建议和模板
- **市场导向**: 结合读者喜好和平台规则给出建议
- **全面覆盖**: 包含创作所需的所有必备选项
